# growth-plan-core

Motor portable para definir protocolos growth, generar tareas iniciales y resumir avance/health trend.

## Qué resuelve

- expone catálogo de planes Foundation, Velocity y Dominance
- genera tareas base por cliente
- resume progreso de ejecución
- calcula tendencia de health score
- alterna módulos activos sin depender de la UI original

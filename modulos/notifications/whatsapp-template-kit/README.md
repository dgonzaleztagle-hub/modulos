# whatsapp-template-kit

Toolkit portable para normalizar teléfonos y construir mensajes o links de WhatsApp sin depender de una API externa.

## Qué resuelve

- normaliza teléfonos a formato `wa.me`
- construye links listos para abrir WhatsApp
- incluye templates reutilizables para cobranza, recordatorios y vencimientos

## Origen

- `CorredoresPro`

## Estado

`usable inicial endurecido`


## Estado real

Ya entrega plantillas y normalización suficiente para uso consistente en flujos de WhatsApp.

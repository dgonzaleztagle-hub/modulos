# scraper-engine-core

Motor portable para discovery de URLs, fetch con timeout y batches con concurrencia limitada.

## Qué resuelve

- prueba variantes `http/https/www`
- normaliza fetch con timeout
- procesa batches con límite de concurrencia
- sirve como base para scrapers multiproveedor

## Origen

- `hojacero`

## Estado

`usable inicial endurecido`


## Estado real

Ya abstrae descubrimiento y fetch concurrente con un contrato suficientemente claro para scraping reusable.

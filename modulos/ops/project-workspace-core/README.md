# project-workspace-core

Motor portable para normalizar el blob operativo de un proyecto interno con clientes, pagos, tareas, notas y credenciales.

## Qué resuelve

- crea un workspace vacío consistente
- normaliza datos parciales o incompletos
- evita `undefined` en tabs operativas
- resuelve actividad reciente entre `createdAt` y `lastModified`

## Origen

- `agendaproyectos`

## Estado

`usable inicial endurecido`

## Estado real

Ya entrega una estructura portable estable para workspaces internos y evita volver a normalizar blobs incompletos en cada proyecto.

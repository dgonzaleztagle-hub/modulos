# project-accounting-core

Motor portable para clientes, pagos y métricas rápidas dentro de un workspace operativo por proyecto.

## Qué resuelve

- calcula ingresos totales
- cuenta clientes activos
- arma registros de clientes y pagos
- soporta cuotas de implementación

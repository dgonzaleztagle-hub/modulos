# corrective-measure-tracker-core

Motor portable para unificar medidas correctivas de distintas fuentes, ordenarlas por plazo y resumir pendientes/vencidas en dashboards operativos.

## Qué resuelve

- normalización de medidas de accidente e inspección
- merge de pendientes desde tablas/fuentes distintas
- orden por deadline
- resumen operativo de pendientes, en proceso, completadas y vencidas

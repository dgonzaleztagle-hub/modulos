# lead-hunting-core

Cliente y utilidades portables para hunting comercial, enriquecimiento y clasificación de leads.

## Qué cubre

- payload de búsqueda
- normalización de leads
- payload de clasificación
- helpers para outreach y puntaje

## Qué no cubre

- llamadas reales a providers
- envío de correos
- scraping directo

## Estado

`usable inicial endurecido`

## Estado real

Ya entrega un contrato portable útil para búsqueda, normalización y puntaje de leads sin depender del provider final.

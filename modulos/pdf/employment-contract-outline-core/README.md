# employment-contract-outline-core

Helpers portables para construir el outline textual de contratos de trabajo y anexos laborales.

## Qué resuelve

- arma secciones base para contrato laboral chileno
- resuelve plazo fijo o indefinido
- genera outline de anexo por ampliación o paso a indefinido
- conserva cláusulas generales y firmantes

## Origen

- `pluscontable.cl/src/lib/contractGenerator.ts`

## Estado

`usable inicial endurecido`


## Estado real

Ya sirve como base reusable para outlines laborales sin acoplar el render final a un proveedor PDF.

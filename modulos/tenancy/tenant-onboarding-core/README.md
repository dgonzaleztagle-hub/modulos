# tenant-onboarding-core

Helpers portables para registro inicial de tenants: slug, validaciones por paso y payload final de onboarding.

## Qué resuelve

- genera slug limpio desde el nombre del tenant
- normaliza slugs editados manualmente
- valida paso de cuenta
- valida paso de empresa/tenant
- arma payload final de registro

## Origen

- `pluscontable.cl/src/pages/TenantRegistration.tsx`

## Estado

`usable inicial`

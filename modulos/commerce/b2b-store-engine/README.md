# b2b-store-engine

Motor portable para catálogo técnico B2B con variantes, carrito cotizable y salida comercial.

## Qué cubre

- variantes por grupo
- cálculo dinámico de precio por modifiers
- carrito con ítems equivalentes por combinación
- resumen y totalización
- mensaje de cotización para WhatsApp

## Qué no cubre

- UI
- persistencia local o remota
- envío real del mensaje
- reglas tributarias o stock avanzado


## Estado

`usable inicial endurecido`


## Estado real

Ya concentra catálogo B2B, variantes y mensaje comercial con un borde reusable muy claro.

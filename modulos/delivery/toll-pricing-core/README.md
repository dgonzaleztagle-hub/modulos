# toll-pricing-core

Helpers portables para estimar peajes desde catálogos de pórticos y una ruta planificada.

## Qué resuelve

- resuelve tramo tarifario por horario y día
- mide cercanía de un pórtico a una ruta
- estima costo total y promedio de peajes por ruta

## Origen

- `acargoo/lib/acargoo/domain/toll-pricing.ts`

## Estado

`usable inicial endurecido`


## Estado real

Ya resuelve peajes y recargos de ruta con un modelo reusable para pricing operativo.

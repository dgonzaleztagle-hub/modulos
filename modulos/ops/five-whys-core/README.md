# five-whys-core

Core portable para modelar, actualizar y resumir análisis de causa raíz con metodología de 5 porqués.

## Qué resuelve

- estructura estándar para `why1..why5` y `root_cause`
- actualización portable de campos
- listado de respuestas efectivas
- resumen para formularios, detalle o exportación

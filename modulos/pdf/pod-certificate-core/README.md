# pod-certificate-core

Helpers portables para construir y renderizar certificados POD de entregas.

## Qué resuelve

- normaliza fechas Santiago y agenda de entrega
- arma filas de certificado POD
- genera PDF base de certificado de entrega

## Origen

- `acargoo/lib/acargoo/domain/pod-pdf.ts`

## Estado

`usable inicial endurecido`


## Estado real

Ya modela el certificado POD con filas y metadata suficientes para generación documental portable.

# inspection-checklist-core

Core portable para inicializar resultados de checklist desde una plantilla, actualizar estados/observaciones y agrupar resultados por categoría.

## Qué resuelve

- creación de resultados base desde una plantilla
- actualización de estado por ítem
- observaciones por ítem
- agrupación por categoría
- resumen de cumplimiento

# zeleri-signature-core

Generador y validador portable de firmas HMAC-SHA256 para payloads de checkout y webhooks.

## Qué resuelve

- ordenamiento estable de objetos y arrays
- concatenación determinística de propiedades
- generación y validación de firma `sha256`

## Origen

- `zeus/zeleri-signature.js`

## Estado

`usable inicial endurecido`

## Estado real

Ya resuelve de forma estable la serialización determinística y la firma/validación HMAC que usan los flujos de checkout y confirmación.

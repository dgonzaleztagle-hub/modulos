# loyalty-ledger-core

Motor portable para calcular puntos, resolver tiers y preparar movimientos de lealtad sin depender del backend final.

## Qué resuelve

- calcula puntos por monto
- resuelve tier por umbrales
- proyecta el siguiente estado del ledger
- arma transacciones portables para award, manual o redemption

## Origen

- `rishtedar`

## Estado

`usable inicial endurecido`

## Estado real

Ya resuelve puntos, tiers y transacciones portables con suficiente estabilidad para no volver a depender del backend original.

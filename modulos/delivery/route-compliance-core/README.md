# route-compliance-core

Motor portable para auditar si una ruta ejecutada respetó la ruta sugerida y sus umbrales operativos.

## Qué resuelve

- compara tracking real vs ruta planificada
- calcula desvío promedio y máximo
- compara duración real contra duración esperada
- infiere si una política rápida tipo TAG fue probablemente usada o no
- devuelve razones revisables para auditoría humana

## Origen

- `acargoo`

## Estado

`usable inicial endurecido`


## Estado real

Ya permite auditar cumplimiento de ruta con criterios portables y cobertura estable.

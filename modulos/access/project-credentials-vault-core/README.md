# project-credentials-vault-core

Vault liviano para asociar credenciales operativas a un proyecto sin depender de la UI original.

## Qué resuelve

- crea un vault vacío consistente
- mezcla credenciales de GitHub, Supabase y Vercel
- agrega, actualiza y remueve credenciales custom
- deja una forma portable para workspaces internos

## Origen

- `agendaproyectos`

## Estado

`usable inicial endurecido`

## Estado real

Ya resuelve un vault portable y reutilizable para credenciales operativas por proyecto sin depender de la UI original.

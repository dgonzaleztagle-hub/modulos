# rental-contract-outline-core

Helpers portables para construir el outline textual de un contrato de arriendo.

## Qué resuelve

- formatea fechas y montos CLP/UF
- arma secciones de partes, inmueble y condiciones económicas
- expone cláusulas base reutilizables

## Origen

- `CorredoresPro/src/components/ContractPDF.tsx`

## Estado

`usable inicial endurecido`


## Estado real

Ya abstrae cláusulas y montos de contrato de arriendo con suficiente estabilidad reusable.

# worker-event-compensation-core

Helpers portables para clasificar eventos de trabajador y calcular impactos económicos en RRHH.

## Qué resuelve

- resuelve metadata de eventos laborales
- calcula valor de horas extras según sueldo, jornada y recargo
- calcula descuentos por atraso y faltas
- resume eventos para trazabilidad o reportes

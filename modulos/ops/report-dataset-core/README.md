# report-dataset-core

Helpers portables para construir datasets y modelos de exportación de reportes de ventas, gastos y productos.

## Qué resuelve

- arma filas de resumen financiero
- normaliza detalle de órdenes y gastos
- prepara detalle de ítems vendidos
- prepara tabla de productos vendidos
- compacta dataset CSV de ventas
- define nombres de archivo consistentes
- entrega modelos listos para workbook/PDF sin amarrarse a una librería específica

## Origen

- `truckpos_new/src/utils/exportReports.ts`

`usable inicial endurecido`

## Estado real

Ya cubre datasets, nombres de archivo y modelos portables para exportación sin casarse con una librería única de Excel o PDF.

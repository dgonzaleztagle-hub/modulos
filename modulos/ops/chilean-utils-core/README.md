# chilean-utils-core

Helpers portables para RUT, UF, CLP y datos base de operación chilena.

## Qué resuelve

- limpia, valida y formatea RUT
- calcula dígito verificador módulo 11
- convierte entre CLP y UF
- formatea montos CLP/UF
- expone catálogo base de regiones de Chile

## Origen

- `pluscontable.cl/src/lib/rutValidator.ts`
- `CorredoresPro/src/lib/chilean-utils.ts`
- `CorredoresPro/src/hooks/useUF.ts`

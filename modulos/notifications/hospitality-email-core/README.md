# hospitality-email-core

Plantillas portables para emails hospitality y food brand, separadas del provider de envío.

## Qué resuelve

- arma correos de confirmación de reserva
- arma bienvenida al newsletter o círculo de marca
- permite personalizar marca, colores y CTA sin depender del repo original

## Origen

- `rishtedar/lib/services/notificationService.ts`

## Estado

`usable inicial endurecido`

## Estado real

Ya tiene templates y personalización suficientes para servir como base de correos hospitality sin depender del proveedor de envío.

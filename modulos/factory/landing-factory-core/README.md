# landing-factory-core

Motor portable para inicializar y manipular configuraciones de landing por secciones.

## Qué resuelve

- inicializa una landing con estructura por defecto
- actualiza bloques y contenido por `sectionId`
- reordena secciones
- lista solo las secciones visibles para render

## Origen

- `hojacero`

## Estado

`usable inicial endurecido`


## Estado real

Ya ofrece un motor portable para estructura y edición seccional de landings sin acoplar el render final.

# sales-agent-core

Cerebro portable para agentes comerciales con perfilado técnico/negocio, objeciones y cierre guiado.

## Qué resuelve

- identifica dolor principal desde una auditoría técnica
- clasifica perfil del usuario entre técnico y negocio
- construye pitch de deseo por contraste
- responde objeciones de presupuesto, confianza y tiempo
- decide cuándo empujar checkout o agenda

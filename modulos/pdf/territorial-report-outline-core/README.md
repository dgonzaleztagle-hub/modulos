# territorial-report-outline-core

Helpers portables para estructurar el outline de un reporte de inteligencia territorial.

## Qué resuelve

- arma portada y metadata base del reporte
- separa secciones de ecosistema, demografía, flujos, competencia, veredicto y digital
- conserva copy y títulos de alto nivel sin depender del renderer PDF

## Origen

- `hojacero/lib/pdf-generator.ts`

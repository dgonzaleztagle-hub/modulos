# outreach-message-core

Helpers portables para personalizar mensajes de prospección, construir HTML de email y resolver links rápidos de contacto.

## Qué resuelve

- reemplazo simple de placeholders
- escape HTML para cuerpos de email
- armado de bloque HTML con firma
- link rápido de WhatsApp y mailto

## Origen

- `bizscout/src/components/leads/LeadDetailModal.tsx`

## Estado

`usable inicial endurecido`


## Estado real

Ya permite generar mensajes comerciales consistentes y reutilizables sin acoplar la salida a un canal único.

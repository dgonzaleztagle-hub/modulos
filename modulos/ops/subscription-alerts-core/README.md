# subscription-alerts-core

Motor portable para detectar vencimientos próximos o expirados y sugerir alertas sin duplicarlas.

## Qué resuelve

- calcula días restantes hacia una expiración
- clasifica alertas en `5 días`, `3 días`, `1 día` o `expired`
- evita duplicar alertas ya emitidas en la misma fecha

## Origen

- `superpanel3.0`

## Estado

`usable inicial endurecido`


## Estado real

Ya tiene suficiente estabilidad para campañas de expiración y deduplicación diaria.

# push-notification-sw-core

Core portable para normalizar payloads de push web y resolver la notificación que debe mostrar un service worker.

## Qué resuelve

- fallback robusto entre `json` y `text`
- defaults consistentes para web push
- resolución del target URL al hacer click

## Origen

- `fidelizacion/public/sw.js`

## Estado

`usable inicial endurecido`


## Estado real

Ya modela payloads y comportamiento de push web con un contrato portable y probado.

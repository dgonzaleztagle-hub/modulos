# download-token-core

Toolkit portable para emitir, normalizar y reutilizar tokens de acceso/descarga con expiración.

## Qué resuelve

- normaliza asset paths de storage
- genera tokens de descarga con expiración
- normaliza email del cliente asociado
- permite reusar o buscar tokens activos en memoria o adapters propios

## Origen

- `zeus`

## Estado

`usable inicial`

# tenant-shell-core

Toolkit portable para resolver branding, prefijos de navegación y estado comercial de un tenant.

## Qué resuelve

- compone branding a partir de tenant y config
- prefija rutas con slug cuando corresponde
- calcula estado comercial y expiración del tenant
- sirve como base para shells privados o públicos por tenant

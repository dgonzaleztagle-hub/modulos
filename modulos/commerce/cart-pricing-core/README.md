# cart-pricing-core

Motor portable para subtotal, descuento, total e inventario básico de un carrito de ordering.

## Qué resuelve

- merge y actualización de items por `menu_item.id`
- cálculo de subtotal
- cálculo de descuento fijo o porcentual
- total final e item count

## Origen

- `rishtedar/lib/store/cart.ts`

## Estado

`usable inicial endurecido`


## Estado real

Ya resuelve cálculos de carrito de forma portable y estable para catálogos, descuentos y totales.

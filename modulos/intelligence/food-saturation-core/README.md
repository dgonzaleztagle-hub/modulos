# food-saturation-core

Motor portable para clasificar oferta food, calcular saturación por categoría y detectar océanos azules/rojos.

## Qué resuelve

- clasifica locales por keywords de cocina
- calcula niveles de saturación por categoría
- resume océano azul y rojo
- sirve como capa analítica sobre scrapers de delivery o maps

## Origen

- `hojacero`

## Estado

`usable inicial endurecido`


## Estado real

Ya resume competencia y saturación gastronómica con una lógica portable para análisis comercial.

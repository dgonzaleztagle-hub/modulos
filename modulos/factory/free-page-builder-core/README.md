# factory/free-page-builder-core

Estado: usable inicial endurecido

## Fuentes

- `superpanel3.0`

## Propósito

Concentrar un builder visual libre tipo canvas para superficies web white-label o comerciales, separado del editor seccional clásico.

## Qué resuelve hoy

- store portable de items, selección, historial y canvas settings
- shell con toolbar, preview desktop/tablet/mobile, inspector lateral y flujo visual tipo lab
- renderer de items libres, presets visuales y widgets enlazables a datos reales
- import/export JSON para puentes con IA, plantillas externas o iteración asistida
- contrato JSON estable para guardar/publicar superficies

## Qué no debe mezclar

- theming o copy específico del reseller
- bloques seccionales rígidos de landing pública
- persistencia acoplada a `resellers.layout_config` como único destino

## Estado real

Ya tiene documento portable, historial, import/export JSON, escalado por dispositivo, widgets con datos reales y un contrato lo bastante claro como para servir de base a un builder visual reusable.

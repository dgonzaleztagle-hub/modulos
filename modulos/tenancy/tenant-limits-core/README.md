# tenant-limits-core

Motor portable para resolver límites operativos por tenant, plan y consumo actual.

## Qué resuelve

- normaliza si un tenant está en plan gratis o pagado
- resuelve el máximo permitido según plan y configuración
- indica si todavía puede agregar más entidades
- sirve para clientes, usuarios u otros recursos contables por tenant

## Origen

- `pluscontable.cl`

## Estado

`usable inicial endurecido`

## Estado real

Ya tiene una base estable para gating de recursos por plan y consumo, reutilizable en varios SaaS del ecosistema.

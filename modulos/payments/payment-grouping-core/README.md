# payment-grouping-core

Motor portable para agrupar pagos consecutivos de una misma entidad y construir metadatos de periodos mensuales.

## Qué resuelve

- agrupa pagos consecutivos por folio
- arma rangos de folio legibles
- consolida meses o periodos cuando corresponde
- sirve para school billing, cuotas o pagos seriados

## Origen

- `kurso`

## Estado

`usable inicial endurecido`

## Estado real

Ya sirve como motor portable de agrupación y resumen de periodos para cobros seriados y school billing.

# booking-rubro-config-core

Configuración portable para adaptar agendas y catálogos de servicios según rubro.

## Qué cubre

- presets de rubro
- normalización de nombre de rubro
- selección de servicios por vertical

## Qué no cubre

- disponibilidad
- reservas o pagos
- gestión de profesionales


## Estado

`usable inicial endurecido`


## Estado real

Ya sirve como capa reusable para adaptar reservas por rubro sin reescribir reglas por vertical.

# worker-registration-link-core

Helpers portables para crear enlaces temporales de registro de trabajador.

## Qué resuelve

- genera token UUID o custom
- calcula expiración por días
- arma URL pública de registro

## Origen

- `pluscontable.cl/src/components/GenerateWorkerLinkDialog.tsx`

## Estado

`usable inicial endurecido`


## Estado real

Ya entrega una base portable confiable para links públicos de onboarding con expiración y tokenización simple.

# promotions-banner-engine

Motor portable para filtrar promociones vigentes, resolver banners activos y preparar payloads de promoción/publicación.

## Qué resuelve

- valida vigencia por fecha, día y hora
- filtra por local o aplica globales
- separa promociones operativas de banners visuales
- selecciona la mejor promoción activa según prioridad de descuento

## Origen

- `rishtedar`

## Estado

`usable inicial endurecido`


## Estado real

Ya resuelve promociones activas y banners con una base portable suficiente para CMS o storefront.

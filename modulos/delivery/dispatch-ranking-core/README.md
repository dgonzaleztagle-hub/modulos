# dispatch-ranking-core

Motor portable para rankear candidatos de despacho según cercanía, disponibilidad, rating y capacidad.

## Qué resuelve

- filtra conductores activos y en turno
- calcula score por distancia, disponibilidad y calidad
- incorpora factor de capacidad/vehículo
- devuelve ranking ordenado para asignación manual o asistida

## Origen

- `acargoo`

## Estado

`usable inicial endurecido`


## Estado real

Ya ofrece un ranking reusable de asignación que puede alimentar dispatch manual o asistido sin depender de UI específica.
